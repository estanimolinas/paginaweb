# GitHub Pages Deployment Instructions

## Step 1: Create Repository
Create a new repository on GitHub at https://github.com/new with the name "paginaweb"

## Step 2: Connect Local Repository to GitHub
After creating the repository, run these commands in your terminal:

git remote add origin https://github.com/estanimolinas/paginaweb.git
git branch -M main
git push -u origin main

## Step 3: Enable GitHub Pages
1. Go to your repository's settings on GitHub
2. Scroll down to the "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click "Save"

Your site will be published at https://YOUR_USERNAME_ORG.github.io/paginaweb/
